Alias a shadow extension

While creating the instance, pass an alias option as a short-hand:

shadow.extend({
    name: 'pindrop',
    alias: 'pindrop',
    template: '<div class="ui-face ui-face--padded">This is an aliased extension</div>'
})

Next, we create the aliased shadow on any element:

<div id="pindrop">This is a pindrop instance.</div>
$('#pindrop').pindrop()
This is a pindrop instance.
